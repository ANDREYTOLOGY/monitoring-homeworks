Задание 1.  
`Вас пригласили настроить мониторинг на проект. На онбординге вам рассказали, что проект представляет из себя платформу для вычислений с выдачей текстовых отчетов, которые сохраняются на диск. Взаимодействие с платформой осуществляется по протоколу http. Также вам отметили, что вычисления загружают ЦПУ. Какой минимальный набор метрик вы выведите в мониторинг и почему?`  

Минимальный набор метрик, которые я включил бы в мониторинг:  

`CPU usage` - потому что вычисления грузят проц  
`RAM usage` - чтобы не упереться в память  
`Disk space` - отчеты пишутся на диск, нужно контроллировать место    
`Disk IO` - чтобы понимать, не тормозит ли запись  
`HTTP метрики (RPS, latency, коды ответов)` - чтобы видеть работу сервиса
